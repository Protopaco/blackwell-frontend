# [011] Reformat Time Sheet Colors

**Board status:** Backlog
**Migrated from:** old `docs/TICKETS.md` "Generated timesheet color palette rework" note (noted 2026-07-21)

## Background

**This is `blackwell-backend` work**, tracked here per the decision to keep one ticket board across both repos.

The generated Google Sheets timesheet's colors have drifted from the original design and need a full pass — both the palette itself and which cells/row-types use which color.

All colors are already centralized in `src/utils/timesheetTheme.ts` (backend repo):
- `PRIMARY` `#0E1733` — structural rows: day/date headers, activity label column, dividers, daily total rows
- `SECONDARY` `#6565AB` — pay period/employee name header rows
- `ACCENT` `#7A305C` — holiday name cells in the holiday header row
- `MUTED_ACCENT` `#D9C7D3` — reserved, unused
- `MUTED_ACCENT_DARK` `#DFB7D2` — holiday columns on odd flat-rate rows
- `MUTED` `#DCDEF0` — weekend/holiday data cells, summary value cells
- `MUTED_DARK` `#CCCCF0` — reserved, unused

Consumed by the various `format*Rows.ts` formatters in `src/services/timesheet/formatting/` (backend repo).

## Scope

Not scoped — needs the target palette (hex values) and the intended cell-to-color mapping worked out before this can be built. That discussion hasn't happened yet.
