# [017] Restyle Pay Period Tabs

**Board status:** Backlog
**Migrated from:** old `docs/TICKETS.md` "Pay Period tabs — visual states" note (noted 2026-07-21)

## Background

The `Tabs` in `PayPeriodLayout.tsx` (Timesheet Status / Payroll Report / Allocation Report) need better visual differentiation between active, clickable/inactive, and disabled states — currently hard to tell them apart at a glance. Disabled state matters here specifically since two of the three tabs are conditionally disabled via `payrollReportGenerated`/`allocationReportGenerated`.

## Scope

Not designed yet — purely a visual/polish pass, no behavior change. Needs a look at the current rendered states before deciding on a specific styling fix (color/weight/underline treatment, etc.).
